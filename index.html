1. Splash Screen

StoormFlow is a tool designed to help Creative professionals eo get inspired and search for ideas

STOORM Giving inspiration through assisted brainstorming process

STOORM Evaluate ideas in a cooperative way

2. Menu

Create a new brief to find the next big idea

View the briefs already created, waiting for team mates to add their ideas and rate them

View the previous finished briefs

3. Notifications

The notifications area shows messages when your are invited to collaborate in other briefs

4. New Briefing

Briefs are based in certain keywords (values associated to the product or its target)

The user has to input some keywords to define the brief, and then the system will use to search the web for ideas

After the user introduced the desired concepts, he/she can press “Start Stoorm” to activate the search

5. Stoorm Process

Guided process to inspire the user exploring divergent ideas encapsulated in a card.

  • 1. Search Ways 10 different types of ideas (i.e. uses and utilities, metaphors, equivalences, daily stories...). If the user swipes over it, the whole card changes to present a new idea
  • 2. Natural phrase The results are shown one at a time
  • 3. Image The tool uses algorithms based on Concepnet to search and retrieve relevant images

To select an idea, the saves it: the card flips and notes can be added on the back

6. End Stoorm

Once the User completes the “Stoorm phase”, his or her brief can be shared with other teammates

When the user decides that there are enough ideas added to the brief, he or she can enter the “Flow phase” to evaluate them

7. Flow Phase

In the Flow phase, the user is presented sequentially with all the added ideas, in random order and mixed with automatically generated ones

The user has to swipe left or right to decide if the idea is good or not. Also, more notes can be added on the back of each idea card

8. End Flow. Ranking

Once the user has finished rating the ideas, this screen shows a ranking by number of votes, and allows to revise the attached notes

The export button generates a pdf containing all this information
